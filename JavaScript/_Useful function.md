#### parseInt()

`parseInt()` function [[converts]] a [[string]] to [[int]].

```js
var a = parseInt("007"); // output: 7
```

If the string can't be converted the [[function]] returns [[NaN]] (Not-a-Number).

You can also [[convert]] it in a specific [radix](https://en.wikipedia.org/wiki/Radix).

```js
var a = parseInt("11", 2);
```

#### trim()

The `trim()` method removes whitespace from both ends of a string.

```js
const greeting = '   Hello world!   ';
console.log(greeting.trim()); // "Hello world!";
```
