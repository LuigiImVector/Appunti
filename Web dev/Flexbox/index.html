Flexbox

I flexbox servono per creare layout flessibili in modo veloce permettendoti di fare cose pazzesche in modo semplicissimo.
❗ Preferisci i flexbox rispetto ai float o position!

Allineamento verticale e orizzontale

justify-content:

  • center
  • flex-start
  • flex-end
  • space-around
  • space-between


align-items:

  • center
  • flex-start (in alto, in caso si lavora in orizzontale)
  • flex-end (in alto, in caso si lavora in orizzontale)
  • stretch (l'item si estenderà per tutta l'altezza del container)
  • baseline

space-around e space-between

Entrambi su utilizzano con la regola justify-content.

space-around: consiste nell'inserire spazio equamente per tutti i lati degli items.
space-between: consiste nell'inserire spazi equamente solamente in mezzo a tutti gli items.

Se vuoi allineare alcune cose a destra e altre a sinistra devi semplicemente raggruppare alcuni items e allinearli a destra (justify-content: flex-end).

Flex-wrap

La regola del flex-wrap è semplicissima: se scrivi flex-wrap: nowrap gli items all'interno del container usciranno fuori se non avranno abbastanza spazio mentre se usi oppure la loro larghezza si restingerà per far stare tutti (quest'ultima ipotesi si verifica quando c'è solo un container flex e non un container flex che contiene il container, scusa per la confusione) flex-wrap: wrap allora gli items in più si posizioneranno sotto in modo da avere una grafica responsive.

Flex-direction e order

La regola flex-direction serve a farti decidere come disporre gli items all'interno di un container, puoi disporli con uno schema a righe (row) oppure a colonne (column), se vuoi decidere quante colonne avere devi usare column-count.

Puoi anche decidere di ordinare gli items al contrario con row-reverse e column-reverse ma se vuoi invece sceglierlo tu l'ordine puoi utilizzare la regola order (devi inserirla nei singoli elementi).

Flex (grow - shrink - basis)

flex: grow | shrink | basis

Questo tipo di proprietà è utile per fare alcune animazioni carine, per esempio quando metti il mouse sopra un item esso si ingrandirà mentre gli altri si rimpiccioliranno [jonasreymondin.com/].

flex-grow: la capacità di quell'elemento di ridimensionarsi se necessario (default value 1);
flex-shrink: la capacità di quell'elemento di ridimensionarsi se il container è troppo piccolo (default value 1);
flex-basis: la dimensione base dell'elemento (default value "auto" | si usano le unità di misura per le lunghezze [px...]).
