Selettori in CSS

Per modificare un elemento HTML tramite CSS esistono 3 modi:

  • Metodo inline: all'elemento HTML si aggiunge l'attributo style e si scrivono le regole CSS;
  • Tag Style: si scrivono le regole CSS aggiungendo il tag <style></style> nel tag head;
  • File esterno: tramite il tag link situato all'interno dell'head si collega un file esterno di CSS al file corrente HTML.
Per i metodi NON inline bisogna usare questi 3 selettori:
  • Classi: si inserisce come attributo nell'elemento (class="nome_classe") e per modificare le regole stilistiche di questi elementi bisogna "richiamarli" con .nome_classe;
  • ID: per modificare tramite gli id bisogna inserire questo attributo in UN SOLO elemento: id="nome_id". Per selezionarli nel file CSS bisogna usare l'hashtag prima del nome dell'ID: #nome_id;
  • Tipologia di elemento: non bisogna aggiungere nessun attributo e per modificare un gruppo di elementi uguali bisogna scrivere il tipo di tag che hanno: p, h1 oppure div. Con questa tecnica si modificano tutti i tag dello stesso tipo; se si seleziona il tag p e si aggiunge la regola color: red; allora TUTTI i paragrafi della pagina avranno il testo di colore rosso (a meno che per qualche paragrafo non si vuole utilizzare un altro colore andando ad inserire un ID, una classe o un metodo inline (magari utilizzando anche !important).


Approfondimenti: InternetingIsHard.

Descendant Selector (selettori discendenti)

Se ti ritrovi con un paragrafo del genere:
<p class="synopsis">
Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod <em>tempor</em> incididunt ut labore et dolore magna aliqua.
</p>
 E definisci questa regola CSS:
.synopsis {
font-style: italic;
}
 Poi ti ritroveresti con tutto il paragrafo in corsivo invece della sola parola 'tempor'.
Per ovviare a questo problema basta usare un selettore discendente:
.synopsis em {
font-style: normal;
}
 Così facendo si aggiunge una regola speciale valida solo per i tag em che sono all'interno della classe synopsis.
N.B. Per selezionare un elemento figlio puoi anche utilizzare: .synopsis > em.

Approfondimenti: MDN, MDN, Udemy, InternetingIsHard.

Pseudo-Classi

Per fare delle piccole "animazioni" con CSS le pseudo-classi sono molto utili. Se per esempio vuoi far cambiare il colore di un pulsante quando passa il mouse sopra esso basta usare la pseudoclasse hover:
.button {
background-color: red;
}
.button:hover {
background-color: blue;
}
 Così facendo quando il mouse passerà sopra il pulsante esso cambierà sfondo da rosso a blu. Per creare qualcosa di più carino puoi anche aggiungere la regola transition.
Altre pseudo-classi sono:

  • :link, un link mai visitato dall'utente;
  • :visited, un link già visitato dall'utente;
  • :active, un link che viene premuto;
  • :first-of-type, il primo elemento della pagina (o di un div se si usano i selettori discendenti);
  • :last-of-type, l'ultimo elemento della pagina (o di un div se si usano i selettori discendenti);
  • :nth-child(2n-1), prende solo alcune righe (in questo caso tutte quelle dispari);
  • :not(.classe), se l'elemento non ha quella classe allora inserisce le regole definite;
  • first-letter, modifica la prima lettera di qualcosa (un paragrafo per esempio);
  • [target], tutti i link che hanno l'attributo target;
  • [href="https://www.google.com"], tutti i link che hanno questo link;
  • [href*="google.com"], tutti i link che hanno nel link questo pezzo (non importa il resto del link, importa che ci sia google.com);
  • [href^="google.com"], tutti i link che iniziano con google.com;
  • [href$="google.com"], tutti i link che finiscono con google.com;
Approfondimenti: MDN, Udemy, InternetingIsHard, W3Schools.
